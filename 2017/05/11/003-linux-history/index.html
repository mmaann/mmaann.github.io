Unix——GNU——Linux

Unix

Unix的前身是由贝尔实验室的Ken Thompson用* 组合语言* 写成的。后来在1971-1973年间由Dennis Ritchie用* C语言* 改写后才成为Unix。
Unix具有可移植性，发展处许多分支，被称为Unix-like的操作系统。
最纯种的Unix：BSD和System V。
1979年：AT&T推出 System V第七版后收回版权。
1986年：x86架构的Minix操作系统诞生。

GNU

1984年:Richard Stallman提出GNU计划，倡导自由软件（free software），并制定发布GPL（General Public Licne）授权模式

Free Software（自由软件）

  • 取得软件与源代码
  • 复制
  • 修改
  • 在发行
  • 回馈

禁止的行为

  • 修改授权：不能将一个GPL授权的自由软件，在修改后取消它的GPL授权
  • 单纯售卖：不能单纯的贩卖自由软件

Open source（开源）授权

  • Apache
  • MIT
  • BSD
  • GPL
  • LGPL

Linux

1991年，芬兰人Linus Torvalds 开发出了Linux操作系统。Linux成功主要归因于：Minix（Unix），GNU， 开源社区等。Linux操作系统最底层的数据成为“核心（Kernel）”

Linux distributions

  • Linux Kernel
  • Free Software
  • Documentations（Tools）
  • 可完整安装的程序

Linux的发型版本

  • 商业版
  • 社区版
  • RPM
    • RHEL
    • Centos
    • Fedora
    • SUSE
  • DPKG（dpt）
    • Debian
    • Ubuntu

Linux的哲学思想

  1. 一切皆文件：把几乎所有资源，包括硬件设备都组织为文件格式；
  2. 由众多单一目的小程序组成：一个程序只实现一个功能，而且要做好
  3. 尽量避免跟用户交互：实现脚本编程，以自动完成某些功能；
  4. 使用纯文本文件保存配置信息：一款使用的文本编程器即能完成系统配置工作。
