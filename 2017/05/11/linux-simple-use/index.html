Linux初识

发表于 2017-05-11 | 分类于 学习 ， Linux运维学习笔记 |

Linux最基本的使用介绍：开机——终端——命令行——帮助——关机

登录

  • 命令行：root/密码(密码输入时不显示)
  • 图形界面：startx

终端

  • 物理终端：直接接入本机的显示器和键盘；/dev/console

  • 虚拟终端:在物理硬件之上以软件方式虚拟实现的终端；centos6、centos7提供6个虚拟终端：tty1-6，Ctrl+Alt+F#[1,6]；/dev/tty1-6

  • 模拟终端：基于ssh或Telnet协议远程打开的界面；/dev/pts/#[0,n]

    查看当前终端设备的命令：tty

交互式接口

启动终端后，附加在终端的交互式应用程序

  • GUI
    • X protocol ,window manager,desktop
    • Desktop
      • Gnome（c，gtk）
      • KDE（c++，gc）
      • XFCE（轻量级桌面）
  • CLI
    Shell程序：
    • sh（bourn）
    • csh
    • tcsh
    • ksh
    • bash（bourn again shell，GPL）
    • zsh

显示当前shell：echo $(shell)
显示系统支持的shell：cat /etc/shells

文件系统简介

文件包括两类数据：

  • 元数据（metadata）：记录文件的属性
  • 数据（data）：记录文件数据
  1. 文件名严格区分大小写
  2. 文件名可使用出/外的任意字符
  3. /表示根目录，也代表目录分隔符
  4. 文件名的长度不超过255个字符
  5. .开头的文件为隐藏文件

路径：

  • 绝对路径：从根目录开始的路径
  • 相对路径：从当前路径起始的路径
    • ..表示当前目录的上一层目录
    • pwd命令显示当前工作目录

Linux命令及帮助

命令：

  1. 文件系统相关
* 目录管理
* 文件管理
* 文件查看
* 文件编辑
  1. 系统管理相关

帮助：

  • 内部命令：help COMMAND

  • 外部命令：

    1. COMMAND -h(--help)
    2. man COMMAND
      手册目录： /usr/share/man
    • man1：用户命令
    • man2：系统调用
    • man3：C库调用
    • man4：设备文件及特殊文件
    • man5：配置文件格式
    • man6：游戏
    • man7：杂项
    • man8：管理类命令

    有些命令会出现在不止一个章节，常看相应章节的文档用：man # COMMAND

    man命令的配置文件：/etc/man.config（centos6），/etc/mandb.conf

    1. info COMMAND
    2. 程序自身文档：README,INSTALL,Changelog
      文档目录：/usr/share/doc/COMMAND-VERSION
    3. 站点官方文档：Documentation
    4. 发行版的官方文档
    5. 其他：搜索引擎，技术论坛

关机

关机：halt，poweroff，shutdown，init 0
重启：reboot，shutdown -r，init 6 查看登陆用户：who，whoami，w
